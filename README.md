# Digitale Strategien Sachsen — Infoportal

Interaktives Infoportal über die sieben digitalen Strategiepapiere des Freistaates Sachsen.

## Funktionen

- **Strategieübersicht**: Alle sieben Strategien mit Prinzipien, Dimensionen und Kernzielen
- **Konflikte & Synergien**: Interaktive Analyse der Wechselwirkungen zwischen den Strategien
- **Reifegrad-Assessment**: Behördenspezifische Selbsteinschätzung mit Radardiagramm

## Deployment auf GitHub Pages

1. Neues Repository auf GitHub erstellen
2. `index.html` hochladen (einzige benötigte Datei)
3. Unter **Settings → Pages → Source** den Branch `main` und Ordner `/ (root)` auswählen
4. Die Seite ist nach ca. 1 Minute unter `https://<username>.github.io/<repo-name>/` erreichbar

## Technologie

Eigenständige HTML-Datei mit React 18 (via CDN). Kein Build-Schritt erforderlich.

## Datengrundlage

Die Inhalte basieren auf der Volltext-Analyse folgender Strategiepapiere:

1. sachsen digital 2030 — Digitalstrategie (2022)
2. Cybersicherheits-Strategie Sachsen (2024)
3. KI-Strategie für den Freistaat Sachsen (2022)
4. Open Source-Strategie der Sächsischen Staatsverwaltung (2023)
5. Innovationsstrategie des Freistaates Sachsen (2023)
6. Strategie der digitalen Transformation im Hochschulbereich (2022)
7. Strategie zur digitalen Transformation der Sächsischen Staatsverwaltung (2023)
